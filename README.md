# Physics Labs Repository

Репозиторий с лабораторными работами по физике от ИТМО (Новый физтех).

## About / О проекте

Сборка лабораторных работ по теоретической и экспериментальной физике от Санкт-Петербургского национального исследовательского университета информационных технологий, механики и оптики (ИТМО), кафедра "Новый физтех".

Каждая работа включает:
- Исходные экспериментальные данные
- Python скрипты для расчётов и обработки данных
- LaTeX отчёты с графиками и результатами

## Labs / Лабораторные работы

- **`lab_4.03/`** — Лабораторная работа 4.03 «Определение радиуса кривизны линзы по интерференционной картине колец Ньютона»
- **`lab_4.14/`** — Лабораторная работа 4.14 «Спектрометр»

## How to Run / Как запустить

See the README in each lab folder for specific instructions.

Инструкции по запуску расчётов и сборке отчётов находятся в README каждой лаборатории.
